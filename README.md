# node-shared

node相关的通用工具

<br />

## features

- pnpm 的
- 开箱即用的
- typescript 的

<br />

## Usage

### install

```shell
pnpm i
```

### test

```shell
pnpm test

// or pnpm test:watch
```

### build

```shell
pnpm build

// pnpm build:watch
```

### coverage

```shell
pnpm coverage
```

### dev

```shell
pnpm dev
```
